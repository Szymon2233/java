Kiedy powstał JavaScript (podaj datę):

JavaScript powstał w 1995 roku. W grudniu tego samego roku został zaimplementowany w przeglądarce Netscape Navigator.

Dlaczego jest to język skryptowy:

JavaScript jest językiem skryptowym, ponieważ jest interpretowany, a nie kompilowany do kodu maszynowego. Do jego uruchomienia wystarczy przeglądarka internetowa.

Gdzie jest wykonywany JS (klient lub serwer):

JavaScript jest wykonywany po stronie klienta, ale dzięki Node.js może być również używany po stronie serwera.

Dlaczego JS to bezpieczne narzędzie:

Ze względów bezpieczeństwa JavaScript ma ograniczony dostęp do zasobów komputera użytkownika. Wszystkie operacje są wykonywane w kontrolowanym środowisku przeglądarki.

Czy jest to język obiektowy, jeśli tak to dlaczego:

Tak, JavaScript jest językiem obiektowym, ponieważ wykorzystuje obiekty, metody i zdarzenia do opisu właściwości i działań. Posiada strukturę opartą na modelu DOM i pozwala tworzyć oraz modyfikować obiekty.
